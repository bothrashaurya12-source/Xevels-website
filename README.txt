
# Xevels — Pre-built Static Website

This is a pre-built static version of the Xevels website (HTML/CSS/JS) containing:
- Home, Shop (2 products), Blog, Contact
- Cart UI with mock checkout
- Images included in the `images/` folder

How to preview locally:
1. Unzip the folder (if zipped).
2. Open `index.html` in your browser (double-click) — works offline.

How to deploy to Vercel:
1. Go to https://vercel.com/new
2. Choose "Upload" and select this folder (or zip and upload).
3. Deploy — Vercel will provide a public link.

Note: This is a static site; for full React features or real payments, use a React project and integrate a backend/payment gateway.
